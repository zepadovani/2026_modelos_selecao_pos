# Estudo transversal de modelos de seleção em PPGs Acadêmicos da área de Música - 2025-26

Este repositório contém os dados e a apresentação visual do levantamento realizado sobre os processos seletivos dos Programas de Pós-Graduação (PPG) em Música no Brasil para o ingresso nos anos de 2025 e 2026.

## Sobre o Estudo

### Objetivos
Este estudo visa mapear e comparar os processos seletivos dos Programas de Pós-Graduação Acadêmicos em Música no Brasil para o biênio 2025-2026, oferecendo uma visão estruturada sobre linhas de pesquisa, vagas, cotas e modalidades de avaliação.

### Metodologia
O estudo foi realizado a partir da simplificação e estruturação de dados de editais publicados entre os anos de 2024 e 2026. O processo de extração utilizou modelos de Inteligência Artificial para simplificação do texto e extração de dados estruturados.

**Aviso Importante:** Devido à natureza automatizada do processo de extração, os dados podem apresentar inconsistências ou omissões pontuais. Recomenda-se sempre a consulta aos editais originais de cada instituição para informações oficiais.
